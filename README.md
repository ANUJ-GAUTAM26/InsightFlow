# InsightFlow

## Overview

InsightFlow is a multi-domain analytics platform that combines:

- YouTube Analytics
- Food Delivery Analytics
- Healthcare Analytics

into a centralized data warehouse and business intelligence solution.

---

## Tech Stack

### Data Engineering

- Python
- PostgreSQL
- SQL

### Analytics

- Power BI

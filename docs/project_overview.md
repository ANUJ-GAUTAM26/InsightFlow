# InsightFlow Analytics Suite

## Overview

InsightFlow is an end-to-end data analytics portfolio project consisting of three independent analytics solutions:

1. Healthcare Analytics Dashboard
2. YouTube Content Analytics Dashboard
3. Food Delivery Analytics Dashboard

The project demonstrates data collection, storage, transformation, analysis, and visualization using modern analytics tools.

## Objectives

* Build professional analytics dashboards.
* Practice SQL data modeling and transformations.
* Develop data visualization skills using Power BI.
* Demonstrate API integration and ETL concepts.
* Create a portfolio suitable for internships and entry-level analytics roles.
